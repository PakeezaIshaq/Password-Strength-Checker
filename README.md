# Password Strength Checker (C++)

A console-based C++ application that evaluates password strength, suggests improvements, and stores hashed passwords securely.

## Features
- Checks password strength (Weak / Medium / Strong)
- Suggests stronger password variations
- Hashes passwords using C++ STL hash
- Saves hashed passwords to file
- Simple console interface

## Technologies
- C++
- STL (string, fstream, functional)
